# Portfolio

This repository contains the personal portfolio website of Roaa Khaldoon, showcasing my background, skills, experience, and selected game development projects.

## Overview

Visit the live portfolio at https://roaahaldoon.net/

The portfolio is a responsive single-page website built with HTML, CSS, and JavaScript. It highlights:

- Personal introduction and professional background
- Technical skills and experience
- Featured projects
- Contact details and links to social platforms

## Featured Projects

The site includes detailed project pages for the following work:

- Animal Farm — an AI-driven simulation featuring autonomous animal behavior
- FSM — a playable simulation that explores finite state machines in game animation
- Reptilian Dungeon — a 2D horror-themed puzzle game developed as a team project

## Technologies Used

- HTML
- CSS
- JavaScript
- GitHub Pages for hosting

## Project Structure

- index.html — main portfolio page
- styles.css — site styling and responsive design
- javaScript.js — interactive behavior for the website
- Projects/ — individual project pages and related assets
- Images/ — banners and visual assets

## How to View Locally

1. Clone the repository
2. Open index.html in your browser, or use a live server extension such as Live Server in VS Code
3. Navigate through the portfolio sections and project pages
